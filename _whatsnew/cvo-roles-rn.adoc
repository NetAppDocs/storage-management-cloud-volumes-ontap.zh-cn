=== 无法执行的活动，按选项卡

下表按选项卡总结了受限活动。

[cols="1,4"]
|===
| 选项卡 | 您无法执行的活动 


| 概述  a| 
* 添加卷
* 添加聚合
* 打开 System Manager
* 使用开/关按钮
* 设置密码
* 更新 ONTAP 版本
* 从工作区中删除系统
* 删除系统
* 使用 *立即升级*
* 从所有功能铅笔菜单编辑 *System tags*
* 从全功能铅笔菜单编辑*计划停机时间*
* 从所有功能铅笔菜单编辑 *Storage classes*
* 从全功能铅笔菜单编辑 *Instance type*
* 从所有功能铅笔菜单中编辑*充电方式*
* 从所有功能铅笔菜单编辑 *License type*
* 从所有功能铅笔菜单编辑 *Write speed*
* 从所有功能铅笔菜单编辑 *Ransomware protection*
* 从所有功能铅笔菜单编辑 *Support registration*
* 从所有功能铅笔菜单编辑 *WORM*
* 从所有功能铅笔菜单编辑 *Data tiering*
* 从所有功能铅笔菜单编辑 *CIFS 设置*
* 从所有功能铅笔菜单编辑 *Route tables*




| 卷  a| 
此选项卡不允许更新卷，您不能添加卷。

* 克隆卷
* 编辑卷设置
* 删除卷
* 创建快照副本
* 从快照副本还原
* 更改磁盘类型（高级选项）
* 更改分层策略（高级选项）




| 聚合  a| 
* 添加聚合
* 添加卷
* 增加容量
* 删除聚合




| 复制  a| 
* 中断复制
* 重新同步复制
* 反向重新同步复制
* 编辑计划
* 编辑最大传输速率
* 更新复制
* 删除复制




| 主画布  a| 
* 添加系统
* 发现系统（适用于所有操作员）
* 使用拖放进行 *复制和同步*
* 使用拖放进行 *Volume caching*
* 使用拖放进行 *Replication*
* 添加卷
* 访问 *优化的成本和性能*
* 打开 System Manager
* 仅当所需的 CVO 权限可用时，才对 Cloud Volumes ONTAP 使用*复制*（不适用于 FSx for ONTAP 或本地系统）
* 仅当所需的 CVO 权限可用时，才对 Cloud Volumes ONTAP 使用*卷缓存*（不适用于 FSx for ONTAP 或本地系统）
* 输入系统
* 使用 AVS 管理按钮




| 无角色  a| 
未分配角色时，将禁用所有操作。

* 访问工作环境
* 访问复制
* 使用 Cloud Volumes ONTAP 创建向导
* 使用复制创建向导
* 了解 Cloud Volumes ONTAP


|===
