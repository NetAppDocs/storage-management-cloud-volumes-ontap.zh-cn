= 使用 NTP 服务器同步 Cloud Volumes ONTAP 系统时间
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
为 Cloud Volumes ONTAP 群集配置网络时间协议 (NTP) 服务器可确保群集时间与网络和外部服务器中的其他系统准确时间同步。将群集时间与外部 NTP 服务器同步有助于保持整个基础架构的一致性。NetApp 默认为新的 Cloud Volumes ONTAP 部署配置 NTP 服务器。但是，如果没有为现有的 Cloud Volumes ONTAP 群集进行配置，则必须配置 NTP 服务器，以便在网络内外进行准确的时间同步。

您可以使用以下命令指定 NTP 服务器：

* https://docs.netapp.com/us-en/bluexp-automation/cm/api_ref_resources.html["NetApp ConsoleAPI"^]。
* ONTAP CLI 命令 https://docs.netapp.com/us-en/ontap-cli//cluster-time-service-ntp-server-create.html["创建集群时间服务 NTP 服务器"^]。



NOTE: 如果您不配置 NTP 服务器，可能会遇到服务中断和时间同步不准确的情况。
